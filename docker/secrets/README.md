# YRVI Docker Secrets — DEPRECATED

> **This file documents the old file-based secrets system, which was replaced in v1.3.0.**
>
> Credentials are now managed through the **secrets container web UI** at `http://localhost:8001` — run `setup_docker.sh --paper` to configure them. Do not create secret files manually.

The `docker/secrets/` directory is kept in the repo as an empty placeholder (all files are git-ignored). The secrets container stores and serves credentials at runtime.

See [FAQ.md](../../FAQ.md) for setup help.
